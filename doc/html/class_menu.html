void Menu::readData ( const std::string &  filename,
bool  header,
bool  label,
bool  nodes,
bool  newgraph 
)

Reads data from a file and initializes the graph.

Time Complexity: O(n), n = number of lines in the file

Parameters
filenameThe name of the file to read data from.
headerFlag indicating whether the file has a header.
labelFlag indicating whether the file has labels.
nodesFlag indicating whether the file represents nodes or edges.
newgraphFlag indicating whether to create a new graph or update an existing one.

This function reads data from a CSV file and initializes the graph based on the file contents. It supports reading both node and edge information. The file format must follow the specified structure, and the data is loaded into the graph object.

Note
If newgraph is set to true, a new graph will be created. Otherwise, the data will be added to an existing graph.
This function assumes that the CSV file is properly formatted and accessible.
