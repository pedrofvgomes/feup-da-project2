{\footnotesize\ttfamily void Menu\+::read\+Data (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{filename,  }\item[{bool}]{header,  }\item[{bool}]{label,  }\item[{bool}]{nodes,  }\item[{bool}]{newgraph }\end{DoxyParamCaption})}



Reads data from a file and initializes the graph. 

Time Complexity\+: O(n), n = number of lines in the file 
\begin{DoxyParams}{Parameters}
{\em filename} & The name of the file to read data from. \\
\hline
{\em header} & Flag indicating whether the file has a header. \\
\hline
{\em label} & Flag indicating whether the file has labels. \\
\hline
{\em nodes} & Flag indicating whether the file represents nodes or edges. \\
\hline
{\em newgraph} & Flag indicating whether to create a new graph or update an existing one.\\
\hline
\end{DoxyParams}
This function reads data from a CSV file and initializes the graph based on the file contents. It supports reading both node and edge information. The file format must follow the specified structure, and the data is loaded into the {\ttfamily graph} object.

\begin{DoxyNote}{Note}
If {\ttfamily newgraph} is set to true, a new graph will be created. Otherwise, the data will be added to an existing graph.

This function assumes that the CSV file is properly formatted and accessible. 
\end{DoxyNote}
